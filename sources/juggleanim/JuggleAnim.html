JuggleAnim Java Juggling Animator

Version 1.4.1 (July 27, 2003)

JuggleAnim has a newer version called Juggling Lab -- check it out!

JuggleAnim is an a Java applet for animating juggling patterns. You can use it to display juggling patterns in your web pages, learn from the lists of pre-defined juggling patterns, and invent new juggling patterns on your own. To use JuggleAnim you need to have a Java 1.1 capable web browser (try Netscape 4.x or IE 4.x) or have access to an applet viewer.

JuggleAnim is open source software, released under the GNU General Public License.

Download JuggleAnim

When JuggleAnim is run as an application from your local system, you have access to additional features such as the ability to save patterns as animated GIF files. To do this, you will need to have an appropriate Java Virtual Machine installed on your system. This is included with Mac OS X; on Windows it's likely that you will need to install Sun's free Java virtual machine.

  • Download everything (including source code) at the Sourceforge project summary page.

Use JuggleAnim at your own site!

We encourage others to use JuggleAnim in their own web pages. Doing this requires just a single JuggleAnim file on your site and a few lines of HTML to make it run. Some of the sites using JuggleAnim today include:

  • JuggleWiki

Want to get involved?

You can contribute to the JuggleAnim project by:

  • Testing and filing bug reports. Submit any bugs to the JuggleAnim bug tracking page. Browse this page to make sure your bug hasn't already been submitted by someone else.
  • Creating new patterns and pattern lists. If you develop new pattern lists, send them along by emailing to one of the admins listed on the project summary page. We'll put up an animator link at the top of this page with your patterns.
  • Joining the project as a developer. If you are a Java programmer, whether fluent or learning, you can join the team and directly contribute to this open-source effort. Send email to one of the project admins listed on the project summary page. You can also browse the JuggleAnim CVS repository anonymously.
